Scripts

Build the interaction tree: an opening line plus outcome-keyed follow-ups. One script drives a door workflow and a text journey.

Opening line
"Hi, I'm a local volunteer chatting to neighbours about the council's solar plan. Have you heard much about it?"
If interested →
"Brilliant. The plan would put panels on the community centre — can we count on your support at the next council meeting?"
If unsure →
"No worries — the short version is it saves the centre about $14k a year. Want me to text you the details?"
If not interested →
"All good, thanks for your time. Have a lovely evening."
